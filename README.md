# nutz-pay
Nutz的支付聚合便捷包

这是个12年挖的老坑，那年搞个三方支付公司只要100W，现在要6个亿，呵呵呵呵。

[电脑网站支付（及时到账）功能](https://doc.open.alipay.com/doc2/detail.htm?spm=a219a.7629140.0.0.Ix6HLx&treeId=62&articleId=103566&docType=1)
